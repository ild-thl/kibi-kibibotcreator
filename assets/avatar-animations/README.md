# Avatar Animations (Lottie JSON)

Place your step-specific Lottie JSON files here.

## File naming convention

The app looks for files by step number:

- `step0.json`  -> Startseite (Schritt 0)
- `step1.json`  -> Schritt 1
- ...
- `step8.json`  -> Schritt 8

If a file for the current step is missing (or fails to load), the app falls back to the existing dummy avatar image.

## Format

Use **Lottie JSON** exported from Adobe After Effects / Lottie tools.
